# Cricket

completely *luck* based cricket made in **rust**

# How to Run

```bash
git clone https://github.com/PremadeS/cricket
cd cricket
cargo run
```

# Note

set your *terminal* size to `120 x 40px` because I'm too *lazy* to make it variable sized
